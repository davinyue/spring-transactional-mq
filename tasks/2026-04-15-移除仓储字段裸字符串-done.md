# 移除仓储字段裸字符串

## 背景

`EzMybatisTransactionalMessageRepository` 中存在 `TABLE.field("updateTime")` 形式的字段裸字符串，不利于重构和编译期检查。

## 需求

- `TABLE.field(...)` 必须使用 Lombok 生成的 `Fields` 常量。
- 当前问题中的 `updateTime` 使用 `BaseEntity.Fields.updateTime`。
- 补充回归测试避免再次引入字段裸字符串。
